== Description ==

This plugin allows you to insert categories to pages too and not only to posts (whitout add new category) . To view the categories in your blog you have to add function to your themes, please see WP the documentation.

== Installation ==

- Download the plugin categories4page
- uncompress ( unzip categories4page.zip )
- copy the file categories4page.php in your plugins directory at your wordpress blog (/wp-content/plugins)
- activate the categories4page Wordpress plugin at your Plugins admin page
